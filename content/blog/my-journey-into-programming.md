# My Journey Into Programming — From Zero to Full-Stack

I remember the first time a browser rendered HTML I had typed. The words appeared on screen and I thought — *I made that happen.* That feeling never gets old.

## The Beginning

I started learning to code in 2022, mostly out of curiosity. I had no mentor, no structured path — just free resources, YouTube tutorials, and a lot of trial and error. freeCodeCamp became my first real curriculum.

HTML was easy to grasp. CSS was frustrating (it still is sometimes 😅). But when JavaScript clicked — when I understood that I could make things *respond* to users — I was completely hooked.

## University Life

Joining Woldia University's Computer Science program gave me structure and peers. Having other people who cared about code made a huge difference. We'd stay up late debugging, sharing resources, and competing on projects.

The curriculum gave me CS fundamentals — algorithms, data structures, operating systems — that made me a better developer, not just someone who follows tutorials.

## The First Real Project

Building the University Grade Management System was the most challenging thing I had done up to that point. Real users. Real data. Real consequences if something broke.

I chose PHP and MySQL because they matched what I was learning in school. The biggest challenge was role-based access control — students, instructors, and admins all needed different views of the same data.

Shipping that project taught me more than any tutorial ever could.

## What's Next

I'm now comfortable with React, Next.js, and building full-stack systems. My goal is to land an internship at a company where I can grow fast, contribute real value, and learn from engineers more experienced than me.
